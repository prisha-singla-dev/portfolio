CodeSentinel

A production multi-agent AI code review platform. Four specialized LangGraph agents read every pull request in parallel — one hunts security holes, one profiles performance, one checks logic, one checks style — and post their findings straight into your GitHub review workflow.

Live demo ↗ GitHub repo ↗ Read the write-up ↗

How it works

  • Four specialized LangGraph agents analyze pull requests in parallel — security vulnerabilities, performance bottlenecks, logical defects, and code quality — each producing structured, review-ready findings.
  • Structured JSON enforcement plus a response-repair step catches malformed model output before it reaches the review, instead of surfacing a broken comment on someone's PR.
  • A cascading fallback chain across seven LLM providers means a single provider outage or rate limit never takes the reviewer offline — it just moves to the next provider in the chain.
  • Findings post directly into GitHub's native PR review flow, so it fits into a workflow engineers already use instead of asking them to check a separate dashboard.
