LLM Cost Firewall

A production API gateway that sits between your app and any LLM provider, cutting inference spend without touching output quality — through smarter routing and aggressive, safe caching.

Live dashboard ↗ GitHub repo ↗ Read the write-up ↗

How it works

  • A six-factor ML complexity model scores every incoming request and dynamically routes it between Gemini 2.5 Flash and Gemini 2.5 Pro — cheap model when the task allows it, capable model when it doesn't — cutting average inference cost by 60%.
  • A dual-layer semantic retrieval system combines SHA-256 exact matching with embedding-based similarity search, catching redundant requests before they ever reach the API.
  • Rate limiting, budget enforcement, and exponential backoff keep the gateway itself from becoming the next production incident.
  • 37 automated unit and integration tests plus a GitHub Actions CI/CD pipeline make it a maintainable service, not a weekend script.
